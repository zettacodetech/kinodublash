# downloads/

Bu papkaga 4-loyihadan (Flutter) build qilingan `app-release.apk` faylini joylashtiring.

Flutter'da build qilish:
```bash
cd ../../4-mobile-flutter
flutter build apk --release
cp build/app/outputs/flutter-apk/app-release.apk ../3-web-frontend/downloads/app-release.apk
```

Shundan so'ng saytdagi "📱 APK yuklash" tugmasi ishlaydi.
